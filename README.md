# Финансовый дашборд · Бизнес-клуб «Вектор»

План vs Факт доходов по направлениям деятельности. Источник — xlsx-файлы в публичной папке Яндекс.Диска.

**Live:** https://suntarme-glitch.github.io/fin-dashboard/

## Что показывает

- KPI: Факт / План / Выполнение % / Отклонение
- График План vs Факт по месяцам / кварталам / году
- Donut: структура факта по направлениям
- Прогресс-бары: выполнение плана по 5 направлениям
- Топ под-направлений (Премиум членство, Конференции, Коучинг и т.д.)
- Heatmap: % выполнения плана по матрице «направление × месяц»
- Tabs: переключение между «Все направления» и каждым из них в отдельности

## Источник данных

Дефолтная публичная папка Я.Диска зашита в `index.html` (`DEFAULT_DISK_URL`). При открытии страницы данные подтягиваются автоматически через REST API:
```
https://cloud-api.yandex.net/v1/disk/public/resources?public_key=<URL>
```
Парсер ищет в имени файла `fact`/`факт` или `plan`/`план` и совмещает данные.

## Формат xlsx

Лист должен содержать строку с заголовками-месяцами (`Январь … Декабрь`). Под ней — строки направлений:
- без отступа в первой колонке = верхний уровень (направление)
- с пробелом в начале = под-направление, относится к последнему направлению выше
- строка `ИТОГО ПО ВСЕМ НАПРАВЛЕНИЯМ` игнорируется

Пример (фрагмент):
```
Направление / Источник дохода | Январь | Февраль | … | ИТОГО
Членские взносы               | 421731 | 365688  | … | 5923586
  Стандартное членство        | 89266  | 70465   | … | 1110135
  Премиум членство            | 99546  | 106989  | … | 1703044
```

## Архитектура

Single-page HTML на GitHub Pages:
- Chart.js — графики
- SheetJS (xlsx.full.min.js) — парсинг Excel в браузере
- Yandex Disk Public REST API — список и скачивание файлов

Без бэкенда. Данные не покидают браузер.
